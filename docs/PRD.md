# PRD: 버스 알리미 봇 (bus-alarm-bot)

## 목표
카카오버스 앱을 켜고 정류장을 검색하는 과정 없이, 텔레그램 명령어 한 번으로 내가 타는 정류장의 버스 도착시간과 혼잡도를 확인한다.

## 사용자
- 1명 (개발자 본인). 매일 같은 출퇴근 경로를 이용한다.
- 아침: 집 앞 정류장에서 직장 방면 버스를 탄다.
- 저녁: 직장 근처 정류장에서 집 방면 버스를 탄다.

## 핵심 기능
1. `/morning` — 아침 정류장(stops.json의 `morning`)에 오는 버스들의 도착시간 + 혼잡도 표시.
2. `/evening` — 저녁 정류장(stops.json의 `evening`)에 오는 버스들의 도착시간 + 혼잡도 표시.
3. routes 필터 — 한 정류장에 오는 여러 버스 중, 내가 타는 노선번호만 골라서 표시.

## 동작 방식 (요약)
- 정류장은 서울 ARS ID(5자리)로 식별한다. **출발 정류장 하나만** 입력하며, 도착 정류장은 입력하지 않는다 (도착시간·혼잡도는 출발 정류장 기준으로 측정되므로 불필요).
- "당산역 가는 버스"는 (올바른 출발 정류장 ARS ID) + (routes에 적은 버스 번호)로 표현된다. 예: 등촌역의 당산역 방면 정류장 ARS ID + `["6516","660"]`.

## MVP 제외 사항
- 경로 탐색 / 길찾기 (어떤 버스를 타야 하는지 봇이 찾아주지 않는다 — 사용자가 이미 안다)
- 도착 정류장 ID 입력, 환승 안내
- 정해진 시간 자동 푸시 (on-demand 명령어만)
- DB / 캐시 / 사용자 계정 (상태 없음)
- 정류장 검색 명령어 (stops.json 수동 1회 설정)
